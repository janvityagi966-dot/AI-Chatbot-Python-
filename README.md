# AI_ChatBot_MiniProject 🤖

A simple Python command-line chatbot that greets users based on current time and answers basic questions using dictionary mapping.

## ✨ Features
- **Time-based Greeting**: Says Good Morning/Afternoon/Evening/Night based on system time
- **Dictionary-based Responses**: Uses Python dict to map keywords to replies
- **Case Insensitive**: Works with `Hello`, `HELLO` or `hello` 
- **Exit Command**: Type `bye` to close the chatbot
- **Beginner Friendly**: Perfect for understanding functions, loops, and dictionaries

**🛠️ Tech Stack** 
• Python 3.x 
• Built-in modules: datetime, time  

**📚 What I Learned**
• Working with datetime module 
• Dictionary key-value mapping 
• String methods: .lower(), .strip() 
• while loops and break statements 
• Function creation and calling
• Debugging with print statements
